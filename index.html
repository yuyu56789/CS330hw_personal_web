Project 1 -- Guess Color

Project 1 -- Guess Color


Technologies include: Color Modeling, Image Cropping, Component Synchronization, UI Interaction, Web Services, and Streamlit.


The main purpose of the software is to obtain data on the perception of the colors of sub-region in different locations in artwork selected by users, and the influence of the surrounding colors on perception. The software operations includes:
1. Start the software;
2. Upload or select a painting;
3. Select a sub-region;
4. Perceive the color of the sub-region;
5. Adjust the slider or picker, so that the color perceived is presented;
6. Submit the color block;
7. Download user data.


Project 2 -- Depth Estimation

Project 2 -- Depth Estimation


Technologies include: GAN Network, Tensorflow, Binocular Vision, Depth Estimation, KITTI, and Jupytor.


The main purpose of the software is to design GAN network, use KITTI data, training for binocular vision effective neural network, and obtain the correct depth information, which can be applied in automatic driving, or industrial detection.
